tplog is a log/trace module for windows platform. Its purpose are:

* Easy to deploy. (e.g. 100% header files)
* Easy to use. To make Tracing become enjoyment.
* Efficient. No unnecessary performance costs.
* Accurate. Micro-second level, sync among different processes.
* Powerful.
    - Abstract log devices which can dynamically add/remove at runtime.
    - Freeform log filter which support compond expressions. (e.g. tag="perf" && level>30)
    - A powerful log viewer. (e.g. filtering, searching, highlighting by expressions, function recognizing, logic visualizations)
